Adopt Me is a pet adoption apps that built with React. It is a part of challenge from [Frontendmasters][1] React course. Adopt me using an online API that provided by [Petfinder][2], a pet adopting company that operates on Unites States. So if you're on the states right now and looking for pet, this apps can help you!ğŸ’¯
